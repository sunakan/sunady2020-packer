= Packer使ってごにょりたい

1. osを選択する
1. git submodule addする


.例：debian
----
$ git submodule add -b master https://github.com/boxcutter/debian.git

== git submoduleを最新にする

$ make update-gitsubmodules
or
$ git submodule update --remote
----
